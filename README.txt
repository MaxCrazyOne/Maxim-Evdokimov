генератор QR-кодов

Андроид-приложение будет представлять собой генератор QR-кодов и будет включать следующие экраны

  -Экран с текстовым полем,QR-кодом и пункатми меню
  -Экран кастомизации QR-кода
  -Экран со сканером QR-кода

На экране кастомизации будет представлена палитра цветов, а также
форма загрузки файла для вставки в код
